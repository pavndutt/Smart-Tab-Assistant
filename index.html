Smart Tab Assistant

Smart tab manager to auto-group inactive tabs, close duplicate tabs, track active tab time, and save tabs for later.

Download Extension v1.0.2

How to install manually:

  1. Download the .crx file above.
  2. Open Chrome and navigate to chrome://extensions
  3. Enable Developer mode in the top right corner.
  4. Drag and drop the downloaded .crx file into the Extensions page.

Future updates will be automatically installed via this GitHub page!
